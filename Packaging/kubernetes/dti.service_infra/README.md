- **Helm 설치**
  
    ```bash
    $ curl -sSL [https://raw.githubusercontent.com/helm/helm/master/scripts/get-helm-3](https://raw.githubusercontent.com/helm/helm/master/scripts/get-helm-3) | bash
    $ source <(helm completion bash)
    $ echo "source <(helm completion bash)" >> ~/.bashrc
    $ echo "source <(helm completion bash)" >> ~/.bash_profile
    $ tail -1 ~/.bashrc
    $ source <(helm completion bash)
    ```
